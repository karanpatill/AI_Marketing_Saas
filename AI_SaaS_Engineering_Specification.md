## Product Vision

Build a production-ready AI SaaS platform that is scalable, secure,
maintainable, reusable, and capable of supporting thousands of users
without major architectural rewrites.

## Engineering Principles

-   Production first
-   Simplicity over complexity
-   Reuse over duplication
-   Security by default
-   Performance by default
-   Strong typing
-   Feature-based architecture
-   Modular design
-   Provider abstraction
-   Clean, readable code

## AI Pipeline

User Input → Project Context → Business Information → Brand Information
→ Brand DNA → Brand Voice → Moodboard → Target Audience → Campaign Goal
→ Asset Type → Context Builder → Prompt Builder → Prompt Optimizer →
Provider Router → AI Provider → Generated Output → Storage → History →
Analytics

## Strict Rules

-   Never generate unnecessary code.
-   Never create unused files.
-   Never duplicate business logic.
-   Never over-engineer.
-   Refactor instead of rewriting whenever possible.
-   Every component must have a single responsibility.
-   Every service must have a single responsibility.
-   Every module must be reusable.
-   Every implementation must be production-ready.
